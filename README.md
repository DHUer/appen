# appen-job task 
全前端Vue项目，未使用后端，词库存放在内存中，每次刷新或者重新打开界面都会删掉词库
## 使用说明
每次输入单词之后，单机右侧的提交按钮，即可更新词库。之后每次输入相近的单词会从之前提交的词库中找到提交次数最多的词，来做为提示词。如下图所示：
